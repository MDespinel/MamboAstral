### Instalacion
#### list
- Clonar repositorio
    - Git clone https://github.com/MDespinel/MamboAstral
- moverse a la carpeta del proyecto
    - cd ecommerce
- instalar las dependencias
    -npm install
- Compilamos
    - npm run dev
